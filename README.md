# [F2E2019] 第十週 - 口罩地圖

## [Demo](https://guanwha.github.io/f2e2019-week10-maskmap/)

## Features
- 根據自身位置搜尋附近的藥局清單
- 點選地址可以前往 google map 查詢
- 可分別過濾成人口罩、兒童口罩、隱藏全數售完的藥局

## Language
- HTML
- CSS/SCSS
- JavaScript

## Framework
- Bootstrap, jQuery
